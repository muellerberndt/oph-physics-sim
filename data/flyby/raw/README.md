# CloseFlyBy Raw OD Replay Area

This directory is intentionally a manifest area, not a bundled raw-data dump.
Historical `CloseFlyBy(F)` certificates remain `OD_REPLAY_PENDING` until real
DSN/ESA/PDS tracking, calibration, solve-for, covariance, residual, and code-hash
artifacts are attached and replayed.

Synthetic OD data may validate the pipeline but must not mark a historical
certificate solved.
