OPH repair-round scale-closure note
===================================

Status: hypothesis / diagnostic lane. This is not a finite proof of 3D bulk,
not a physical CMB prediction, and not a derivation of exactly 24 repair
rounds. It is useful because it links the local P closure, the global N_CRC
capacity closure, and the observed scalar-tilt target in one compact scale law.

Maarten's equation:

  |g'(P)| = N_CRC^(-1/48)

Interpretation:

  g is the local repair / pixel-closure map near the OPH fixed point P,
  so that delta_{n+1} = g'(P) delta_n.

  The exponent 48 is interpreted as 2 * 24. The factor 2 appears because
  holographic capacity scales like area. The 24 is a declared repair-depth
  hypothesis until a finite selector theorem fixes it.

If one repair round expands length scale by q = 1 / |g'(P)|, then after
24 rounds:

  q^24 ~ R / l_P

Squaring gives capacity:

  q^48 ~ N_CRC

so:

  |g'(P)| = q^-1 = N_CRC^(-1/48)

OPH local-pixel form:

  |g'(P)| = alpha(P) / phi^2
          = (P - phi) / (sqrt(pi) phi^2)

and therefore:

  N_CRC = |g'(P)|^-48

Numerically, with P ~= 1.6309682094039593:

  alpha(P) / phi^2 ~= 0.002787
  q ~= 359
  N_CRC(P) ~= 4.3e122
  n_s = 1 - P/48 ~= 0.9660215

Practical simulation consequence:

  m_eff(N_sim) = ln(N_sim) / (-2 ln |g'(P)|)

Under this depth model, 64k, 256k, and 1M literal-patch regulators represent
only about one logical scale-repair round. The fix is not to expect raw patch
count scaling to reach cosmic behavior. The simulator needs an explicit
scale-compressed 24-round repair operator or a finite theorem/certificate that
derives the equivalent logical scale rounds.
